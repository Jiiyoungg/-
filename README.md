# 자율주행 센서의 안테나 성능 예측
데이콘 : 자율주행 센서의 안테나 성능 예측 AI 경진대회


배경
=============
Radar는 자율주행 차에 있어 차량과의 거리, 상대 속도, 방향 등을 측정해주는 필수적인 센서 부품으로, 제품의 성능 평가 공정에서 양품과 불량을 선별하고 있음.
 -> AI 기술을 활용하여 공정 데이터와 제품 성능간 상관 분석을 통해 제품의 불량을 예측/분석
